How to use the programm:

1. Download the Java File (only takes a few seconds)

2. Open the HitTheBox.txt file inside TINY RISC STUDIO

3. Click compile and start an emulation.

How to play the game:

- Control the Sprite with the buttons on the bottom on the CPU

- Try to fit the sprite in the middle of the hitbox (hitbox is 2px bigger)

- Once you are in the middle the hitbox will recompute its coordinates using a random number generator
